# front-restart
## Goal

このリポジトリの目的は、
フロントエンドの基礎技術を身につけ、
在宅で仕事を受けられるレベルに到達することである。

小規模でも、
自分で作成したWebページを説明でき、
修正や改善に対応できる状態を最終目標とする。

## Roadmap

まずはHTMLとCSSを使って、
静的なWebページを一人で作れるようになる。

次に、JavaScriptを用いて、
ボタン操作や簡単な動きを実装できる状態を目指す。

その後、簡単なページ制作や修正作業を想定し、
実務に近い形での練習を行う。

## Required Skills

このリポジトリで扱う主な技術は以下の通りである。

HTML：ページの構造を作成するために使用する。
CSS：文字色やレイアウトなど、見た目を調整するために使用する。
JavaScript：ユーザー操作に応じた動きを実装するために使用する。

これらの技術は、practiceフォルダで実際に動かしながら習得する。
